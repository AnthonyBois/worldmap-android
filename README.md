## Procédure d'installation

1. Récuperer le projet depuis GIT

```
git clone https://github.com/AnthonyBois/worldmap-android.git
```

2. Se placer dans le dossier du projet et installer les dépendances

```
cd WEB-worldmap
npm install

```
